# Pytubefix-GUI
Provides a nice graphical user interface to use the pytubefix library to download files from Youtube.

# Installation

### Create Virtual Environment
```
python -m venv .
```

### Activate
```
.\Scripts\activate
```

### Install Libraries
```
pip install -r requirements.txt
```

# Usage

1) Activate your virtual environment

2) Run the script.
```
python main.py
```
